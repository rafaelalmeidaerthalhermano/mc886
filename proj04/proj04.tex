\section{Choosing K}
\subsection{Intracluster Variance}
To calculate the intracluster variance, the following function was implemented:

\begin{lstlisting}
import numpy as np
import math

def intracluster_variance(data, clusters):
    size = len(data)
    sum  = 0
    for i in range(size):
        point   = data[i]
        cluster = clusters.labels_[i]
        center  = clusters.cluster_centers_[cluster]
        sum += np.linalg.norm(center - point)
    return math.sqrt(sum / size)
\end{lstlisting}
